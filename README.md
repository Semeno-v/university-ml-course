# 📊 Учебные проекты по классическому машинному обучению (Classical ML)

Данный репозиторий содержит решения лабораторных работ по курсу машинного обучения. Фокус проектов направлен на классические алгоритмы (без использования Deep Learning), разведочный анализ данных (EDA), конструирование признаков и работу с временными рядами.

## 🛠 Стек технологий
* **Язык программирования:** Python
* **Обработка и анализ данных:** Pandas, NumPy
* **Машинное обучение:** Scikit-learn, Statsmodels
* **Визуализация:** Matplotlib, Seaborn
* **Получение данных:** yfinance

---

## 📂 Содержание курса (Вариант №3)

### Лабораторная работа 1: Кластеризация и анализ метрик
* **Данные:** Breast Cancer Dataset (`sklearn.datasets`).
* **Задача:** Разделение объектов на кластеры по базовым признакам (`radius`, `texture`) с последующим добавлением `perimeter`.
* **Методы:** K-Means.
* **Оценка качества:** Метод «локтя», расчет средних силуэтных коэффициентов, оценка Accuracy (Acc) при сравнении с фактическим разбиением.

### Лабораторная работа 2: Исследование факторов и логистическая регрессия
* **Данные:** Статистика государственных закупок лекарственных препаратов (ЕИС 2021).
* **Размер выборки:** 75 000 процедур (Train/Test: 95% / 5%).
* **Задача:** Предсказание вероятности незакрытия закупки (закупка не состоялась).
* **Методы и этапы:**
  * Конструирование признаков (Feature Engineering), включая метод главных компонент (PCA) / факторный анализ.
  * Агрегация данных до уровня «одна строка — одна закупка».
  * Оценка важности признаков (Feature Importance) с помощью алгоритма **Decision Tree (Дерево решений)**.
  * Построение и валидация модели **Логистической регрессии**.

### Лабораторная работа 3: Анализ и прогнозирование временных рядов
* **Данные:** Исторические котировки акций **Tesla (TSLA)** (интервал: 1 неделя).
* **Задача:** Прогнозирование цен закрытия на 10 будущих периодов.
* **Методы и этапы:**
  * Парсинг финансовых данных (Yahoo Finance).
  * Оценка стационарности ряда (Тест Дикки-Фуллера), анализ АКФ и ЧАКФ.
  * Grid Search гиперпараметров для модели **ARIMA(p, d, q)**.
  * Выбор оптимальной модели по критерию **AIC** и оценка качества прогноза с помощью MAPE.
